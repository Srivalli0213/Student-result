# Student Result Calculator (Python)

This is a simple Python project that calculates a student's total marks, average, and grade based on subject marks.

# Features
- Takes marks as input
- Calculates total and average
- Displays grade (A/B/C/D/Fail)
- Simple and beginner-friendly Python code

# How to Run
1. Install Python (if not installed)
2. Download this project or clone:
   git clone https://github.com/Srivalli0213/Student-result.git
3. Run the file:
   python student_result.py

# Example Output
Enter marks of 5 subjects:  
Total = 420  
Average = 84  
Grade = A
